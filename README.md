# Diamod-Price-Predictor
This project presents an interactive Diamond Price Prediction Dashboard built 
using Streamlit, designed to perform end-to-end machine learning operations in a 
structured and user-friendly way. 
The dashboard follows a step-by-step pipeline including data upload, EDA, data 
cleaning, feature selection, data splitting, model training, evaluation, and tuning. 
Users can upload a housing dataset in CSV format, after which the system provides 
key insights such as total records, number of features, and feature types, along 
with visualizations for better data understanding. The project focuses on 
regression-based prediction of house prices using various input features like 
location, income, and housing characteristics, while also displaying model 
performance through evaluation metrics
